# 📊 Pulse Bot

Pulse Bot is a simple Python automation project that fetches weather data and sends a daily summary via email using GitHub Actions.

---

## 🚀 Features

- 🌦️ Fetches real-time weather data (OpenWeather API)
- 📧 Sends daily email summary
- ⚙️ Fully automated using GitHub Actions
- 🔐 Secure API keys using GitHub Secrets

---

## 🧠 Tech Stack

- Python 🐍
- OpenWeather API 🌦️
- GitHub Actions ⚙️
- SMTP (Gmail Email) 📧
